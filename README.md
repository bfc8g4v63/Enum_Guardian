# EnumGuardian

## 功能簡介

* USB VID / PID 掃描與即時監控  
* 自動清理重複與殘留 COM Port  
* 監控裝置插拔事件（IPEVO、Castles、FTDI、Silicon Labs 等驅動）  
* 提供 GUI 與 CLI 兩種模式操作  
* 支援批次解除鎖定與重新安裝驅動  
* 可匯出設備清單報表（Vendor、Product、Driver、COM 編號）
